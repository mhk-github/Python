# ImageSelector  

Presents a listbox to select an image file to display.  

**Files:**
- *cache.py*
  - Image file cache system
- *ImageSelector.py*  
  - Python 3 script
- *imageview.py*
  - PyQt5 image file viewer
- *selectionview.py*
  - PyQt5 image file selection window
- *settings.py*  
  - Configuration file
