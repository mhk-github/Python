# B3D  

A selection of utilities for working with binary B3D files of 3D data.  

**Files:**  
- *AnalyzeB3DFile.py*  
  - Show information about B3D files  
- *B3DUtilities.py*  
  - Used by all scripts dealing with B3D files  
- *ObjToB3D.py*  
  - Converts a Wavefront OBJ text file into a B3D file
