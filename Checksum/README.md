# Checksum

A Python 3 PyQt5 application to check file checksums.

**Files:**
- *Checksum.py*
  - The driver script for this project
- *Checksum_GUI.py*
  - The PyQt5 GUI version of this project
- *lint.txt*
  - Flake8 information
- *requirements.txt*
  - Python 3 modules needed
- *settings.py*
  - Settings file for this project
